# BetterAuth Service

Passwordless authentication using Better Auth SDK.

## Tech Stack
- ASP.NET Core
- Better Auth SDK integration
- PostgreSQL (shared with Identity)

## Port
5130

## Dependencies
- Dhadgar.ServiceDefaults
- Dhadgar.Contracts

## Notes
- Handles OAuth flows (Google, GitHub, Discord, etc.)
- Session management
- Integrates with Identity service for user records
