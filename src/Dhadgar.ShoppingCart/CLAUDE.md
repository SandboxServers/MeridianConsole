# ShoppingCart - Marketing & Checkout

Marketing site and subscription checkout flow.

## Tech Stack
- Astro 5.x
- React
- Tailwind CSS

## Commands
```bash
npm install
npm run dev      # Dev server on port 4322
npm run build    # Production build
```

## Key Directories
- `src/pages/` - Astro pages (login, pricing, etc.)
- `src/components/` - React components (OAuth buttons, etc.)
- `src/lib/auth/` - Authentication client

## Dependencies
- @dhadgar/shared-auth (local package)

## Notes
- Deploys to Azure Static Web Apps
